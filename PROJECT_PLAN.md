# 🌌 LIGHTYEARS — Project Plan (PRD)

> **Tagline:** "Tonight, a star is showing you the day you were born. Go find it."
> Yeh document tera roadmap hai. Confuse ho to wapas yahin aa.

---

## 1. Project kya hai (1 line)
User apni birthday daalta hai → website use ek **real star** dhoondh ke deti hai jiski roshni
us din nikli thi jab woh paida hua. Ek beautiful galaxy mein woh star dikhta hai, aur
user us star pe ek message "pin" kar sakta hai.

**Why people love it:** Personal + emotional + real science + screenshot-worthy.

---

## 2. Kis cheez se banega (Tech Stack)
| Cheez | Tool | Kyun |
|---|---|---|
| Page banana | HTML + CSS + JavaScript | Sabse basic, beginner-friendly |
| 3D galaxy | Three.js | Stars ko 3D mein dikhane ke liye |
| Star ka data | HYG Database (CSV → JSON) | ~119,000 REAL stars, free, no API key |
| Daily space photo | NASA APOD API (DEMO_KEY) | Roz wapas aane ka reason |
| Hosting (free) | GitHub Pages ya Vercel | Project online dikhane ke liye |
| Messages (baad mein) | Supabase free tier | Doosron ke notes dikhane ke liye |
| Share card | html-to-image | Instagram/TikTok pe share = viral = ratings |

---

## 3. PHASES — chhote tukdon mein banayenge
> Har phase apne aap mein "shippable" hai. Matlab har phase ke baad project online daal sakte ho.

### ✅ PHASE 1 — "Find Your Star" (sabse pehle, ~1-2 din)
- Ek simple page: birthday daalne ka box + button
- Button dabane pe: HYG data se woh star nikalo jiska distance ≈ user ki umar (light-years)
- Ek sundar card dikhao: star ka naam, constellation, distance, rang
- **Yahi ship karenge sabse pehle.** Stardust shuru. 🎉

### 🌟 PHASE 2 — 3D Galaxy (~3-5 din)
- Three.js se asli star-field banao (background mein hazaaron stars)
- Camera user ke star pe zoom kare
- Mouse se ghuma/zoom kar sako

### 🛸 PHASE 3 — "Universe is watching" (breakthrough feature)
- Timeline slider: "Abhi universe Earth ka kaunsa saal dekh raha hai?"
- Historical events (moon landing, world cup) ko stars se map karo
- Yeh wala feature viral hota hai

### 📌 PHASE 4 — Starposts (message pin karna)
- User apne star pe ek note chhod sake
- Share-card auto-banao (PNG download) → social media

### 🌍 PHASE 5 — Social layer
- Doosron ke public notes dikhao, ❤️ do
- Daily NASA photo
- Galaxy insano ke messages se jagmagati hai
